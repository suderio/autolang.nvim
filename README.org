* Autolang.nvim

A lightweight, pure Lua plugin for Neovim that automatically detects the natural language of the current buffer and adjusts the =spelllang= option accordingly.

** Features

-  *Zero Dependencies*: Written in Pure Lua. No Python, Node, or external binaries required.
-  *Fast*: Uses a "Stop Words" heuristic (word frequency) on the first 50 lines of the buffer.
-  *Support*: English, Portuguese, Spanish, French, German, and Chinese.
-  *Configurable*: Supports interactive mode (ask before changing) and dialect mapping (e.g., pt -> pt_BR or pt_PT).

** Installation

*** Lazy.nvim (Recommended)

#+BEGIN_SRC lua
{
  "suderio/autolang.nvim", -- Replace with local path or git repo
  event = { "BufReadPost", "BufWritePost" },
  config = function()
    require("autolang").setup({
        -- Your custom config here (optional)
                             })
  end
}
#+END_SRC

** Configuration

The default setup works for most use cases. Below are all available options:

#+BEGIN_SRC lua
require("autolang").setup({
    -- Enable auto-detection on buffer open (BufRead) and save (BufWrite)
    auto_detect = true,

    -- Interactive Mode:
    -- false: Changes spelllang silently (default)
    -- true: Opens a prompt asking if you want to change the language
    interactive = false,

    -- Performance limit: Number of lines to read from the start of the file
    lines_to_check = 50,

    -- Limit the languages to check against.
    -- Useful if you only write in specific languages (e.g., English and Portuguese).
    -- Set to nil to check all available languages.
    limit_languages = { "en", "pt" },

    -- Mapping of detected language code -> Vim's 'spelllang' option
    -- This is where you differentiate dialects (e.g., pt_BR vs pt_PT, en_US vs en_GB)
    lang_mapping = {
        en = "en_us",
        pt = "pt_br", -- Change to 'pt_pt' for European Portuguese
        es = "es",
        fr = "fr",
        de = "de",
        zh = "cjk",
    },
})
#+END_SRC

** Commands

- =:AutolangDetect= : Forces a manual detection on the current buffer.
